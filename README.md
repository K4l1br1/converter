# Unit Converter

Конвертер единиц измерения - консольная утилита для конвертации различных единиц измерения.

## Возможности

- Конвертация расстояний (метры, километры, мили, футы и т.д.)
- Конвертация веса (килограммы, граммы, фунты, унции и т.д.)
- Конвертация температуры (Цельсий, Фаренгейт, Кельвин)

## Сборка

### Обычная сборка

```bash
make
```

### Сборка deb пакета

Для сборки deb пакета выполните:

```bash
./build_deb.sh
```

Это создаст файл `unit-converter_1.0.0_amd64.deb`, который можно установить командой:

```bash
sudo dpkg -i unit-converter_1.0.0_amd64.deb
```

Или через apt:

```bash
sudo apt install ./unit-converter_1.0.0_amd64.deb
```

## Использование

После установки запустите программу:

```bash
unit_converter
```

## Удаление

Для удаления пакета:

```bash
sudo apt remove unit-converter
```

## Требования

- g++ (компилятор C++)
- libc6
- libstdc++6
- dpkg-deb (для сборки deb пакета)

**Тесты**

- **Расположение**: исходники тестов находятся в файле `tests/test_runner.cpp`.
- **Что проверяется**: конвертации расстояний (`DistanceConverter`), веса (`WeightConverter`) и температуры (`TemperatureConverter`), включая типичные случаи, граничные значения и проверку генерации исключений при неизвестных единицах.
- **Сборка и запуск**: тестовый раннер компилируется с помощью `Makefile` и создаёт бинарник `test_runner`.

Для сборки тестов и получения бинарного файла выполните:

```bash
make test
```

```bash
./test_runner
```
